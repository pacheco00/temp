17. Use a RHEL system role
Create a playbook name selinux.yml and use system roles
Set selinux mode as enforcing in all manage node
====
cat /usr/share/ansible/roles/rhel-system-roles.selinux/README.md
grep -n -C 3 "enforcing" /usr/share/ansible/roles/rhel-system-roles.selinux/README.md
====


[admin@control ansible]$ vim selinux.yml

- name: Configure selinux as enforcing mode
  hosts: all
  vars:
    - selinux_state: enforcing 
  roles:
          - role: rhel-system-roles.selinux
            become: true
: wq!

[admin@control ansible]$ ansible-navigator run selinux.yml -m stdout
